# Изучение Git

| Команда | Что делает |
| :--- | :--- |
| 'git init' | Вариант Б для Автоматизации |
| 'git add' | Добавляет изменения для последующего коммента |
| 'git commit' | Сохраняет изменения в репозиторий |
| 'git push' | Отправляет изменения в репозиторий |
| 'git pull' | Получает изменения из удаленного.репозитория и объединяет их с локальным |
